In der course.yaml-Datei muss eine Seite mit Unterseiten definiert werden. Folgende Seitentypen stehen dabei zur Verfügung:

steps-large
Ablaufschritte mit einer Maximalbreite von 1280 px
steps-medium
Ablaufschritte mit einer Maximalbreite von 960 px
steps-small
Ablaufschritte mit einer Maximalbreite von 480 px
steps-fullwidth
Ablaufschritte, welche die volle Fensterbreite nutzen
steps-fullscreen
Ablaufschritte, die zusätzlich bis an den unteren Fensterrand gestreckt werden
