Each step corresponds to an HTML page. The page type only draws the step list above the content. In order for the pages to optically fit, they should be designed as spectre.css cards: 
<div class="card">
    <div class="card-header">
        <h4 class="card-title">Card Heading (optional)</h4>
    </div>

    <div class="card-body">
        Description text of the card
    </div>

    <div class="card-image">
        <!-- Wide picture, source code example, ... -->
        <img src="%content-url%/img/my-picture.jpg" alt="My Picture" style="width: 100%" />
    </div>

    <div class="card-footer">
        <!-- Navigation buttons -->
        <subpage-nav-buttons></subpage-nav-buttons>
    </div>
</div>
