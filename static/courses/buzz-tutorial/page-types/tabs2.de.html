In der course.json-Datei muss eine Seite mit Unterseiten definiert werden. Folgende Seitentypen stehen dabei zur Verfügung:

tabs-large
Tabreiter mit einer Maximalbreite von 1280 px
tabs-medium
Tabreiter mit einer Maximalbreite von 960 px
tabs-small
Tabreiter mit einer Maximalbreite von 480 px

Und so sieht die course.json aus: 

---
language: ""
name: "Buzz Tutorial"
pages:
    page-types/tabs/:
        name: Per Tabreiter umschaltbare Unterseiten
        type: tabs-large
        pages:
            1:
                pos: 1
                name: So sieht es aus
                file: page-types/tabs1.de.html
            2:
                pos: 2
                name: Beispiel für course.yaml
                file: page-types/tabs2.de.html
            3:
                pos: 3,
                name: Inhalt der HTML-Dateien
                file: page-types/tabs3.de.html
    ...
