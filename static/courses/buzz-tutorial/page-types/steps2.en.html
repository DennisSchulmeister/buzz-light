In the course.yaml files sub-pages must be defined. The following page types are available for that:

steps-large
Steps with a maximum width of 1280 px
steps-medium
Steps with a maximum width of 960 px
steps-small
Steps with a maximum width of 480 px
---
language: ""
name: "Buzz Tutorial"
pages:
    page-types/steps/:
        name: Step-For-Step Sequences
        type: steps-large
        pages:
            1:
                pos: 1
                name: That's How It Looks
                file: page-types/steps1.en.html
            2:
                pos: 2
                name: course.yaml Example
                file: page-types/steps2.en.html
            3:
                pos: 3,
                name: Content of the HTML Pages
                file: page-types/steps3.en.html
    ...
