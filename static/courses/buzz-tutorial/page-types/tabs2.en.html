In the course.yaml files sub-pages must be defined. The following page types are available for that:

tabs-large
Tab pages with a maximum width of 1280 px
tabs-medium
Tab pages with a maximum width of 960 px
tabs-small
Tab pages with a maximum width of 480 px

And this is how course.json looks: 

---
language: ""
name: "Buzz Tutorial"
pages:
    page-types/tabs/:
        name: Tabbed Sub-Pages
        type: tabs-large
        pages:
            1:
                pos: 1
                name: That's How It Looks
                file: page-types/tabs1.en.html
            2:
                pos: 2
                name: course.yaml Example
                file: page-types/tabs2.en.html
            3:
                pos: 3,
                name: Content of the HTML Pages
                file: page-types/tabs3.en.html
    ...
