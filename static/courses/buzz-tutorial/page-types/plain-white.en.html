Description

Course pages contain no background of their own by default, so that the website background is fully visible. However this can make pages with a lot of text hard to read. Thus each page may be assigned one or more classes in the course.yaml file in order to influence their background.

The following classes can be used:

white
Bright background with dark font
black
Dark background with bright font
padding
Adds a little space around the text so that it doesn't stick so much at the border

course.yaml Example

The background is set like this:

--- language: "" name: Buzz Tutorial pages: page-types/plain/white/: name: Simple Page With White Background type: grid-large file: page-types/plain-white.en.html classes: - white - padding ...
