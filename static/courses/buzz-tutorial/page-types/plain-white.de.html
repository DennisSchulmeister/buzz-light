Beschreibung

Standardmäßig werden alle Seiten ohne eigenen Hintergrund dargestellt, so dass das Hintergrundbild der Webseite komplett sichtbar ist. Wenn eine Seite viel Text beinhaltet, kann dieser dadurch aber schwer lesbar sein. Deshalb gibt es in der course.yaml die Möglichkeit, jeder Seite durch Verwendung einer der folgenden Klassen einen Hintergrund zu geben:

Folgende Klassen können den Seiten zugeordnet werden:

white
Heller Hintergrund mit dunkler schrift
black
Dunkler Hintergrund mit heller Schrift
padding
Fügt einen kleinen Randabstand hinzu, damit der Text nicht so sehr am Rand klebt

Beispiel für die course.yaml

Der Hintergrund wird wie folgt festgelegt:

--- language: "" name: Buzz Tutorial pages: page-types/plain/white/: name: Einfache Seiten mit weißem Hintergrund type: grid-large file: page-types/plain-white.de.html classes: - white - padding ...
