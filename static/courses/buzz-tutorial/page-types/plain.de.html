Beschreibung

Dies ist der einfachste Seitentyp. Es handelt es sich einfach um eine HTML-Seite mit beliebigem Inhalt. Die Gestaltung ist deshalb aber auch am schwierigsten, da man auf großen Bildschirmen mit der Breite klar kommen muss, ohne dass der Text zu sehr auseinander gerissen wird.

Zum Glück gibt es hierfür verschiedene Seitentypen unterschiedlicher Größe. Insbesondere die Grids können bei der Gestaltung helfen.

plain
Leere Seite mit maximaler Breite (Dies ist der Standardseitentyp)
grid-large
Zentriertes CSS-Grid mit einer Maximalbreite von 1280 px
grid-medium
Zentriert CSS-Grid mit einer Maximalbreite von 960 px
grid-small
Zentriert CSS-Grid mit einer Maximalbreite von 480 px

Beispiel für die course.yaml

Und so sieht die course.yaml aus: 

---
language: ""
name: Buzz Tutorial
pages:
    page-types/plain/:
        name: Einfache Seiten ohne Template
        type: grid-large
        file: page-types/plain.de.html
    ...
