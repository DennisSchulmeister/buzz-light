<assign-words-quiz>
  • Dies ist das Hauptelement des Quiz, in dem alle anderen Elemente enhalten sind.
  • Optional können dem Tag style und class-Attribute hinzugefügt werden, um das Aussehen des Quiz zu beinflussen.
<quiz-title>
  • Legt den Namen des Quiz fest.
<quiz-description>
  • Beinhaltet eine kleine Beschreibung mit Hinweisen für den Spieler.
  • Die Beschreibung kann durch beliebige HTML-Tags formatiert werden.
<quiz-answers name="all-answers">
  • Definiert eine Liste mit möglichen Antworten. Der Name der Liste wird später verwendet, um sie den Bildern zuzuordnen.
  • Jedes Quiz muss daher mindestens eine Liste mit Antworten enthalten.
  • Innerhalb dieses Elements sind nur <quiz-answer">-Elemente erlaubt.
<quiz-answer name="dog" amount="2">Hund</quiz-answer>
  • Definiert eine einzelne Antwort innerhalb einer <quiz-answers>-Liste.
  • Der Name der Antwort wird benötigt, um die Antwort mit einem Bild zu verknüpfen.
  • Fehlt das amount-Attribut, kann die Antwort nur einem Bild zugeordnet werden. Andernfalls definiert das Attribut, wie oft die Antwort zulässig ist.
<quiz-feedback-positive>
  • Die hier definierte Nachricht wird am Ende des Spiels angezeigt, wenn alle Antworten korrekt sind.
  • Die Nachricht kann mit beliebigen HTML-Elementen formatiert werden.
<quiz-feedback-negative>
  • Wie <quiz-feedback-positive>, nur dass die Nachricht immer dann angezeigt wird, wenn mindestens eine Antworte falsch ist oder fehlt.
<quiz-row>
  • Dieses HTML-Tag definiert eine Reihe mit einem oder mehreren Bildern (Karten).
  • Innerhalb das Elements sind daher nur <quiz-card>-Tags erlaubt.
  • Jedes Quiz muss mindestens eine <quiz-row>-Element mit mindestens einer <quiz-card> besitzen.
<quiz-card answers="all-answers" correct="dog happy">
  • Definiert ein einzelnes Bild (eine sog. Karte), dem der Spieler die richtigen Begriffe zuordnen muss.
  • Welche Antworten zur Auswahl stehen, wird durch das answers-Attribut bestimmt. Es verweist auf den Namen eines <quiz-answers>-Elements.
  • Welche Antworten die richtigen sind, wird durch das correct-Attribut festgelegt. Es enthält eine durch Leeerzeichen getrennte Liste mit den Namen der Antworten.
  • Der Inhalt der Karte kann mit beliebigem HTML-Code formatiert werden.
  • Optional können die Attribute style und class verwendet werden, um das Aussehen der Karte zu beeinflussen.
  • Damit die Karten einheitlich erscheinen, sollte der Inhalt aller Karten exakt gleich hoch sein. Bilder müssen daher identische Proportionen haben.
