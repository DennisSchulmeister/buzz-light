<assign-words-quiz>
  • This is the main element of the quiz, which contains all other elements.
  • Optionaly the style and class attributes can be used to influence the quiz's look.
<quiz-title>
  • Sets the name of the quiz.
<quiz-description>
  • Contains a small description with hints for the player.
  • The description may be formated with arbitrary HTML tags.
<quiz-answers name="all-answers">
  • Defines a list with possible answers. The name of the list is used to assign it to the pictures.
  • Therefor each quiz must contain at least one list of answers.
  • This tag may only contain <quiz-answer"> tags.
<quiz-answer name="dog" amount="2">Dog</quiz-answer>
  • Defines a single answer inside a <quiz-answers> list.
  • The name of the answer is needed in order to link it to a picture.
  • If the amount attribute is missing, the answer can only be assigned to one picture. Otherwise the attribute sets how often the answer can be used.
<quiz-feedback-positive>
  • This message here is shown at the end of the game if all answers are correct.
  • The message may be formated with arbitrary HTML tags.
<quiz-feedback-negative>
  • Just like <quiz-feedback-positive> except that the message will be shwon if at least one answer was wrong or missing.
<quiz-row>
  • This HTML tag defines a row with one or more pictures (cards).
  • Thus this tag may only contain <quiz-card> elements.
  • Each quiz must have at least one <quiz-row> element with at least one <quiz-card>.
<quiz-card answers="all-answers" correct="dog happy">
  • Defines a single picture (a so-called card), to which the player must assign the correct words.
  • Which answers can be selected is determined by the answers attribute. It refers to the name of a <quiz-answers> tag
  • Which answers are the right ones is determined by the correct attribute. It contains a space-separated list with the names of the answers.
  • The card's content may be formated with arbitrary HTML code.
  • Optionaly the attibutes style and class can be used, to influence the card's look.
  • In order for all cards to appear the same, their content should have exactly the same height. Thus pictures must have the same aspect ratio.
