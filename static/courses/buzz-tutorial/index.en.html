Available Page Types

Simple Pages Without a Template
This is the most simple page type. Simply a bit of HTML without anything else. Plain and simple.
Screen-filling Pages
Special occaisions need special foramts. If you like to design your presentation or other graphical content in an aesthetically pleasing manner this page type could be something for you.
Tabbed Sub-Pages
Here you see how you can divide a page into several sub pages between which the user may switch at the top of the screen.
Step-For-Step Sequences
This page type is best suited for step-for-step tutorials. With a few simple HTML tricks you can create beautiful process descriptions.
Chapters Like a Book
A man, a word. A woman, a dictionary. In case you have to say a bit more you may create book-like chapters for your pages.


Page Contents

Course Pages Are Plain HTML
I guess you thought so. But just to be safe, course pages are simple HTML fragments which give you maximum freedom in the realization of your ideas.
Formatted Text via Markdown
If all you want is to write some text with a few simple formattings (bold, italic, underlined, enumerations etc.) you don't need to use HTML at all. Markdown is much easier to write and read, instead.
Syntax Highlighted Source Codes
What would be a programing course without source code examples? Creating these is actually quite simple, including automatic syntax highlighting.
Tiles on the Start Page
Did you notice? This page is completely made with tiles. In order for you to also get this look you'll learn here which HTML code it takes.
Responsive Grid Elements
Besides simple tiles you can align almost any content along a responsive CSS grid. This gives you total layout control for small and large screens alike. No matter what the page will always remain usable.
What to Think About for Pictures
A picture is worth a thousand words. Including pictures is acutally quite simple thanks to HTML. But for them to be equally usable on small and large screens alike you need to consider a few simple rules.
Elements Defined by Spectre.css
Buzz uses the astonishing spectre.css framework. Therefor you can use everthing offered by spectre.css yourself. And this is acutally quite a lot.
