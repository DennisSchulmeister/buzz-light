Larger Blocks of Source Code

Integrating source code examples into a page is quite easy. All you need is the <source-code> HTML element:

<source-code language="javascript" filename="factorial.js"> function factorial(n) { if (n > 1) return n * factorial(n - 1); else return 1; } </source-code>

The result looks like this:

function factorial(n) { if (n > 1) return n * factorial(n - 1); else return 1; }
language
Sets the programming language for the syntax highlighting
filename
Sets the name of the source file that will be shown in the top-right

Within a Line

If you simply want to highlight a small area within a line or an enumaration you can use the <src-code> element, instead. Basicaly it also highlights the source code but doesn't render a new block.

The syntax <src-code language="javascript">if (…) { … } else if (…) { … } else { … }</src-code> allows to check conditionals in JavaScript.

The syntax if (…) { … } else if (…) { … } else { … } allows to check conditionals in JavaScript.


Beware: Special Characters

Since the source code is simply pasted into the page's HTML code, you need to replace any special character not allowed in HTML. Especially the following characters are affected, even when you want to show HTML-code itself.

Special Character   Replacement
& &amp;
> &gt;
