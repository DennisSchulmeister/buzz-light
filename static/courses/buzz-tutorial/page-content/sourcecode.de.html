Größere Quellcode-Blöcke

Quellcodeausschnitte lassen sich ganz einfach in eine Seite integrieren. Alles, was man dazu braucht, ist das <source-code> HTML-Element:

<source-code language="javascript" filename="factorial.js"> function factorial(n) { if (n > 1) return n * factorial(n - 1); else return 1; } </source-code>

Das Ergebnis sieht dann wie folgt aus:

function factorial(n) { if (n > 1) return n * factorial(n - 1); else return 1; }
language
Legt die Programmiersprache für das Syntax Highlighting fest
filename
Legt den Namen der Quellcodedatei fest, der oben rechts angezeigt wird

Innerhalb einer Zeile

Will man innerhalb einer Zeile oder einer Aufzählung nur einen kleinen Bereich einfärben, kann hierfür das <src-code>-Element verwendet werden. Es bewirkt ebenfalls eine Einfärbung des Quellcodes, stellt diesen jedoch nicht in einem eigenen Block dar.

Die Syntax <src-code language="javascript">if (…) { … } else if (…) { … } else { … }</src-code> erlaubt es, Bedingungen in JavaScript abzuprüfen.

Die Syntax if (…) { … } else if (…) { … } else { … } erlaubt es, Bedingungen in JavaScript abzuprüfen.


Achtung: Sonderzeichen

Da der Quellcode ganz einfach in den HTML-Code der Seite eingefügt wird, muss man darauf achten, in HTML nicht erlaubte Sonderzeichen zu ersetzen. Dies betrifft insbesondere die folgenden Zeichen, auch dann, wenn man HTML-Code selbst darstellen will.
